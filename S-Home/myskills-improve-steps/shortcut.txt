PHPでまとめて echo する、外部テンプレート化する、関数化する の3つが代表的な「1つのコードで自動生成」する方法です。

✅ 方法1：PHPの関数でヘッダーを一括自動生成する（最もシンプル）
<?php
function load_header($title = "Profile Site") {
    echo <<<HTML
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>{$title}</title>
    <!-- 自動でCSS/JS読み込み -->
HTML;

    load_css_js(); // ←あなたの関数を呼び出す

    echo <<<HTML
</head>
<body>
HTML;
}
?>
使うとき
<?php
load_header(); // これだけでOK
?>




✅ 方法2：ヘッダー部分をテンプレートファイル化して include する
header.php を作る：
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title><?php echo $title; ?></title>
    <?php load_css_js(); ?>
</head>
<body>
使うとき：
<?php
$title = "Profile Site";
include "header.php";
?>






✅ 方法3：完全自動生成するクラスを作る
<?php
class Template {
    public static function header($title = "Profile Site") {
        ?>
        <!DOCTYPE html>
        <html lang="ja">
        <head>
            <meta charset="UTF-8">
            <title><?= $title ?></title>
            <?php load_css_js(); ?>
        </head>
        <body>
        <?php
    }
}
?>
使い方：
<?php Template::header(); ?>

ChatGPT:
このコードは、シンプルなHTMLテンプレート に、PHP の load_css_js() 関数を組み込んだものですね。
具体的には：

<!DOCTYPE html>：HTML5の宣言（ブラウザにHTML5文書であることを伝える）
<html lang="ja">：日本語を指定したHTML文書
<head>：メタ情報や外部CSS/JSを読み込む部分
<meta charset="UTF-8">：文字コードをUTF-8に設定（日本語を扱うためには必須）
<title>My Theme Practice</title>：ブラウザのタブに表示されるタイトル
<?php load_css_js(); ?>：PHPで定義された関数 load_css_js() を呼び出して、スタイルシートやJavaScriptを読み込む
<body>：HTMLの本文部分（このコードでは空）
これをEmmetで自動生成したい場合
このコードを VS CodeのEmmetスニペット として一発で生成する方法を教えます！
Emmetスニペットを作成する方法
VS Codeの設定を開く


Ctrl + Shift + P → 「Preferences: Configure User Snippets」を選択。
html.json を選択（もしくは新規で作成）。
以下のスニペットを追加：
{
    "HTML + PHP Template with load_css_js()": {
        "prefix": "htmlphp", 
        "body": [
            "<!DOCTYPE html>",
            "<html lang=\"ja\">",
            "<head>",
            "    <meta charset=\"UTF-8\">",
            "    <title>${1:My Theme Practice}</title>",
            "    <?php load_css_js(); ?>",
            "</head>",
            "<body>",
            "    ${2}",
            "</body>",
            "</html>"
        ],
        "description": "HTMLテンプレート + PHPのload_css_js()読み込み"
    }
}
使い方
新しいHTMLファイルで htmlphp と入力し、Tab キーを押すと、自動でこのテンプレートが展開されます。
生成されるコード
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>My Theme Practice</title>
    <?php load_css_js(); ?>
</head>
<body>
</body>
</html>
まとめ
Emmetのスニペット を使うと、このようなテンプレートを数文字で生成でき、開発の効率がかなり上がります。
このテンプレートでは、基本的なHTML5構造に加えて、PHPでCSSやJSを読み込む処理 を組み込んでいます。
もし他にも追加したい内容やカスタマイズしたい部分があれば、教えてください！
